# Car-Price-Prediction
Welcome to the Car Price Prediction Model repository! This project aims to build a car price prediction model using machine learning techniques.
The dataset used for this project is included in the repository.

## Dependencies
Before running the code in this repository, make sure you have the following dependencies installed:
- Python 3.x
- Pandas
- Scikit-Learn
- Matplotlib
- Seaborn

## Data Preprocessing
Before building the model, the dataset is preprocessed. The steps include:
- Checking for missing values
- Label encoding to convert textual columns into numeric columns

## Exploratory Data Analysis (EDA)
EDA is performed to understand the data better.
Some of the EDA tasks performed in this project include:
- Displaying dataset shape
- Showing statistical info about the dataset
- Finding correlation between features
- Plotting categorical data counts

## Model Building
The dataset is split into input data (X) and label data (Y). 
It is then further split into training and testing data. 
Two different regression models are built and evaluated.

## Linear Regression
A Linear Regression model is created and trained using the training data.
The model is used to make predictions on both the training and testing data. 
The Linear Regression model achieves an accuracy of 80% on test data.

## Lasso Regression
A Lasso Regression model is created and trained using the training data.
Similar to Linear Regression, predictions are made on both the training and testing data. 
The Lasso Regression model achieves an accuracy of 76% on test data.

## Conclusion
This project provides a simple example of car price prediction using machine learning techniques.
It covers data preprocessing, exploratory data analysis, and model building.
By analyzing the accuracy scores, you can evaluate the performance of both the Linear Regression and Lasso Regression models in predicting car prices.
